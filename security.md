# Security Policy

## Supported Versions

This project is a personal portfolio website.  
Only the latest version on the `main` branch is actively maintained.

| Version | Supported |
|--------|-----------|
| Main (latest) | ✅ |
| Older versions | ❌ |

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

### How to Report
- Please **do not** open a public GitHub issue.
- Send details via email to: **chiragparashar1812@gmail.com**
- Include:
  - Description of the issue
  - Steps to reproduce (if applicable)
  - Potential impact

### Response Timeline
- Initial response within **48 hours**
- Valid issues will be reviewed and addressed as soon as possible
- You will be informed whether the report is accepted or declined

Thank you for helping keep this project secure.
